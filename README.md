
# img_gallery

The app is made using ReactJs and Material design .
You can search any image of your choice , view it , and clicking on it  will open a modal where you can view the entire image .
## Features
- Search , View , Expand
- Cross platform
- Responsive

  
## Run Locally

Clone the project 

Go to the root directory
```bash
  npm install
```
```bash
  npm start
```
